# Open-Source Boundary

This skill separates **raw references** from **distilled knowledge**.

## Safe to commit
- Style cards (`*.yaml`) with abstract parameters
- Transfer briefs (`*.md`) with generic constraints
- Rubrics, schemas, templates, and scripts
- Non-identifying aggregate notes

## Do not commit
- Third-party raw screenshots, PDFs, or design files
- Near-replica redraws that can reconstruct original assets
- Large copied text from external pages/papers
- Source files with unclear rights

## Review checklist before commit
- No raw external assets included
- No direct reproduction of unique graphic elements
- Distilled outputs are generic and reusable
- Attribution/license needs are respected when applicable

## Practical rule
Open-source the **method and abstractions**, not the **source asset library**.
