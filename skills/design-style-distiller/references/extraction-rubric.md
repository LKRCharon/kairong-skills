# Extraction Rubric

Score each dimension with: `low | medium | high` plus a one-line note.

## 1) Layout
- Information density
- Whitespace generosity
- Grid strictness
- Visual center stability

## 2) Hierarchy
- Primary vs secondary contrast
- Heading/body/caption separation
- Annotation prominence

## 3) Typography
- Weight contrast
- Size scaling consistency
- Label readability at publication size

## 4) Color system
- Number of active colors
- Muted vs vivid tendency
- Accent strategy (single accent / multi-accent)
- Contrast safety for print/screen

## 5) Shape and lines
- Line thickness tendency
- Corner roundness tendency
- Marker/icon style restraint
- Border/shadow usage

## 6) Figure semantics
- Legend placement and compactness
- Axis and tick minimalism
- Label verbosity level

## 7) Tone and domain fit
- formal / playful / technical / editorial
- publication-ready vs prototype-like

## Output discipline
After scoring, always produce:
1. `impression` keywords
2. `dos` (3-8 rules)
3. `donts` (3-8 rules)
4. one transfer brief paragraph
